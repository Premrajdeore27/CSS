What is CSS?

  1. CSS Stand for Cascading Style Sheet.

  2. CSS describes how HTML elements are to be displayed on screen,paper or in other media

  3. Css is not Neccesary CSS saves a lot of work. It can control the layout of multiple web pages all at ones

  4. External stylesheet are store in CSS files.


CSS can be added to HTML Documents in 3 ways:

  • Inline - by using the style attributes inside HTML elements1
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS. is to keep the styles in external CSS files. However, in this tutorial we will use inline and internal styles, because thise is easier to demonstrate, and easier for you to try it yourself

When comparing CSS class vs ID, the diffrence is that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique elements. ID is also special in you can used a special URL to link directly to an element and its used by javascript.


In CSS, selector are used to target a specific element or range of elements on a web page. Once an elements has been targeted, a style or set of style can be applied to the elements.


There is a wide range of selector availables.Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied
